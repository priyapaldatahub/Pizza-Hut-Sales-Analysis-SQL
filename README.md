# 🍕 Pizza Hut Sales Analysis Using SQL

## 📌 Project Overview

This project analyzes Pizza Hut sales data using SQL to uncover business insights related to revenue, orders, pizza categories, and customer purchasing patterns.

The analysis uses joins, aggregate functions, subqueries, Common Table Expressions (CTEs), and window functions to answer real-world business questions.

---

## 🎯 Objectives

- Analyze total sales performance
- Identify top-selling pizzas
- Calculate revenue generated by each pizza type
- Analyze order trends and customer behavior
- Perform category-wise sales analysis
